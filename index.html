diepvries documentation¶

This is the homepage for the documentation of diepvries, a Data Vault framework written in Python.

By automatically generating SQL statements for a given Data Vault model, diepvries is a convenient library to manage data loading without duplicating many SQL queries.

diepvries is free software, released under the MIT license. You can check its source code here. Bug reports and contributions are encouraged!

Head over the topics below for more information, or dive directly in the tutorial if you’re eager to write code already!
